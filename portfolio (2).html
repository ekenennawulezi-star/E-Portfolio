Cybersecurity — Offensive Security

Vulnerability Assessment & Penetration Testing Lab

Built a virtual lab environment running both Windows and Linux virtual machines to simulate a real-world network for security testing. Used Nmap to perform host discovery and port scanning, identifying open services and potential attack surfaces. Applied Metasploit to test known CVEs against the virtual targets, successfully exploiting vulnerabilities and documenting the steps taken. Wrote a full findings report covering each vulnerability discovered, its severity, and a recommended remediation for each — mirroring the deliverable expected in a real penetration testing engagement.

Skills demonstrated: Nmap, Metasploit, virtual machine setup, vulnerability documentation, remediation recommendations.

Cybersecurity — Defensive / SOC

Network Traffic Analysis & Incident Triage

Analyzed PCAP (packet capture) files using Wireshark to identify suspicious network behavior and indicators of compromise (IOCs). Identified anomalous traffic patterns including unusual outbound connections, unrecognized protocols, and signs of potential data exfiltration. Produced written incident analysis reports documenting each finding, the evidence trail, and the recommended response actions — the same core workflow performed by a Tier 1 SOC analyst during an active investigation.

Skills demonstrated: Wireshark, PCAP analysis, IOC identification, incident report writing, SOC analyst workflow.

Networking & Network Security

Secure Network Architecture Design

Designed a multi-segment secure network topology in Cisco Packet Tracer, incorporating subnetting, VLAN segmentation, and firewall rule configuration. Applied defense-in-depth principles by layering access controls at the perimeter, between segments, and at the host level. The design addressed real-world security requirements including restricting inter-VLAN traffic, controlling inbound and outbound flows, and isolating sensitive systems. Also completed traceroute analysis assignments tracing traffic across international providers including Charter/Spectrum, Akamai, and NORDUnet, documenting routing behavior and latency patterns.

Skills demonstrated: Cisco Packet Tracer, subnetting, VLAN design, firewall rules, defense-in-depth, traceroute analysis.

Compliance & Technical Writing

HUD-Compliant Tenant Notices & Policy Documents

As part of my internship with the Suffolk Redevelopment and Housing Authority, I drafted a full set of HUD-compliant legal and operational documents for use across three public housing properties. Documents produced include lease termination notices, formal grievance procedure forms, tenant entry and work-authorization notices, and pre-REAC inspection resident notices — all sourced directly from SRHA's Admissions and Continued Occupancy Policy (ACOP) and aligned with HUD's NSPIRE inspection standards. Each document had to meet specific regulatory language requirements while remaining clear and readable for tenants. This work required understanding federal housing policy, translating regulatory requirements into plain language, and producing documents on official agency letterhead.

Skills demonstrated: HUD/NSPIRE compliance, ACOP policy interpretation, legal document drafting, technical and regulatory writing.

System Administration

Since May 2026, I have been supporting IT operations as a System Administrator Intern at SRHA, a HUD-regulated public housing authority serving the City of Suffolk, Virginia. Responsibilities include day-to-day systems support across multiple office locations, coordinating an internet service provider migration from Spectrum to Glo Fiber while ensuring continuity of a long-standing agency email address, and maintaining data systems used across three housing properties (Colander Bishop Meadows, Hoffler Apartments, and Chorey Park). This internship has provided hands-on experience in a real IT environment with live systems, regulated data, and operational accountability — not a lab simulation.

Skills demonstrated: Systems administration, ISP migration coordination, IT operations support, HUD-regulated environment, multi-property data management.

Data Systems & Operations

Multi-Property Data Organization & Reporting

Built and maintained a version-controlled Excel workbook tracking outstanding tenant balances and payment tier classifications across three housing properties, with a structured change log maintained between each revision. Additionally, transcribed and reorganized a multi-unit handwritten maintenance inspection list covering over 30 units at Hoffler Apartments into a categorized, filterable Excel workbook with three tabs: one sorted by issue category (electrical, plumbing, mold, refrigerator gaskets, smoke detectors, windows), one sorted by unit number, and a summary matrix showing issue counts per unit. This system made property-wide maintenance planning significantly faster and gave management a clear view of where the heaviest repair loads were concentrated.

Skills demonstrated: Microsoft Excel, data organization, version control, operational reporting, multi-property data management.

Independent Project & Technical Documentation

2012 Dodge Durango Citadel — Digital Modernization Build

Independently researching and documenting a full digital retrofit of a 2012 Dodge Durango Citadel AWD as a content and documentation project. The build targets a gap in existing online resources: almost no documentation exists for modernizing the 2012–2013 Durango specifically. Research areas include CAN bus PID mapping and OBD-II compatibility for a CANavie 12.3″ digital instrument cluster swap, CarPlay/Android Auto integration, 2014+ racetrack LED tail light conversion, and exhaust and powertrain upgrades for the 5.7L Hemi. This project demonstrates the ability to independently research complex technical systems, synthesize information from fragmented sources, and produce structured documentation — the same skill set applied to IT systems work.

Skills demonstrated: Independent technical research, CAN bus / OBD-II systems, structured documentation, hardware integration planning.
